Welcome to Vatic-Gurobipy's documentation!
==========================================
``vatic-gurobipy`` is a Python package of building and implementing
Unit Commitment and Economic Dispatch models using gurobi.

Check out :doc:`main_model_functions_gurobi` to use vatic-gurobipy main functions to
generate the models and solve them.

Check out :doc:`construct_models_gurobi` and :doc:`transmission_gurobi` to look at the detailed codes of building mathematical models behind
Unit Commitment and Economic Dispatch.
